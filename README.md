# Проект: Агентство недвижимости (Бриф 1)
## Предсказание стоимости жилья

1. **Описание проекта**

    **Бизнес-задача**: Разработать сервис для предсказания стоимости домов на основе истории предложений. Риелторы тратят слишком много времени на оценку объектов и поиск выгодных предложений, что снижает конкурентоспособность агентства. Точная модель оценки позволит ускорить работу и повысить качество сделок.

    **Техническая задача**: Разработать модель машинного обучения для предсказания цены недвижимости с высокой точностью. Проект включает полный цикл Data Science: от обработки сырых данных до развертывания модели.

    **Данные**: Датасет `data.csv.zip` содержит историю предложений (характеристики объектов, цены).

2. **Цели проекта**

    *   Провести разведочный анализ данных (EDA), очистить датасет от выбросов и пропусков.
    *   Спроектировать новые признаки (Feature Engineering) для улучшения качества модели.
    *   Построить Baseline-модель и сравнить несколько алгоритмов ML (Linear Regression, Random Forest, Gradient Boosting).
    *   Добиться максимальной метрики качества (выбор метрики будет обоснован в проекте, например MAE/MAPE).
    *   Подготовить отчетность и код, соответствующий стандартам (PEP 8, воспроизводимость).

3. **Этапы работы**

    1.  **Первичная обработка**: Загрузка, очистка, обработка дубликатов и пропусков.
    2.  **EDA**: Анализ распределений, корреляций, статистические тесты.
    3.  **Feature Engineering**: Работа с категориальными признаками, генерация новых фичей.
    4.  **Моделирование**: Обучение моделей, подбор гиперпараметров, кросс-валидация.
    5.  **Анализ результатов**: Интерпретация модели (Feature Importance), выводы.

4. **Используемые технологии**

    *   **Python**: Основной язык.
    *   **Pandas/Numpy**: Обработка данных (с использованием GPU ускорения cudf/cupy где возможно).
    *   **Matplotlib/Seaborn/Plotly**: Визуализация.
    *   **Scikit-learn/CuML**: Машинное обучение.
    *   **Scipy**: Статистический анализ.

5. **Оборудование**
    *   Проект выполняется с использованием GPU-ускорения (NVIDIA RTX 4060 Laptop GPU) в среде WSL Ubuntu.

**Примечание**: Решение опирается на собственные вычисления, но учитывает референсные подходы для обеспечения качества.
